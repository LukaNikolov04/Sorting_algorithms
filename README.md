C# aplikacija razvijena sa ciljem implementacije i direktnog upoređivanja performansi tri različita algoritma za sortiranje: Bubble Sort, Quick Sort i Radix Sort. Projekat obuhvata programiranje samih algoritama, izvršavanje operacija nad nizovima podataka i merenje tačnog vremena njihovog izvršavanja (benchmarking) kako bi se praktično analizirala i uporedila njihova vremenska složenost.
